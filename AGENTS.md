# Hermes News Reader

## Project Overview
A Flutter mobile app that consumes structured news data generated by the Hermes news pipeline on a Raspberry Pi 4B. Replaces infinite-scroll Telegram delivery with a proper reader: rich article cards with images, swipeable sections, searchable history.

**Data store:** GitHub repo `pencrusader/trivia__news` (public). Pi pushes JSON via git; phone fetches from `raw.githubusercontent.com`.

**Target:** Android sideload APK → iOS + App Store later.

## Tech Stack
| Layer | Technology | Why |
|-------|-----------|-----|
| App | Flutter 3.x (Dart) | iOS + Android from one codebase |
| State | Riverpod | Type-safe, testable |
| HTTP | `http` package | Simple REST calls |
| Images | `cached_network_image` | CDN images with disk cache |
| Markdown | `flutter_markdown` | Briefs are markdown |
| Data store | GitHub raw (JSON files) | Free CDN, zero infrastructure |
| Pi scripts | Python 3.11 (in venv) | Already running |
