# Research — Hermes News Reader

> Store research findings, competitor analysis, papers, articles, and reference material here.
> Each entry is a dated markdown note. Use this as your project's research notebook.

## Format
Create files as: `YYYY-MM-DD <topic>.md`

## Example entries
- `2026-06-29 Competitor Analysis.md` — Feature comparison of 3 competitors
- `2026-06-29 API Research.md` — Notes on Stripe vs Paddle for payments
- `2026-06-30 Deployment Options.md` — Vercel vs Fly.io vs self-host

## Tips
- Link to source URLs
- Tag with `#research #<topic>` for Obsidian graph view
- Move actionable findings to `planning/DECISIONS.md` after team review
